# 变压器
符号：B或T
## 用途
电压变换，阻抗变换，隔离，稳压，隔直流
## 分类
| 分类方法      | 类别 |
|--------------|---|
| 冷却方式      | 干式(自冷)，油浸式，氟化物 |
| 铁心或线圈结构 | 芯式，壳式，环型，金属箔   |
| 铁心形状      | E型，C型，环型           |
| 防潮方式      | 开放式，灌封式，密封式     |
| 用途         | 电源，调压，音频，中频，高频，脉冲，隔离，自耦，恒压，耦合 |
| 电源相数      | 单相，三相，多相         |

## 低频变压器
电源变压器，音频变压器
## 中频变压器
单调谐式，双调谐式
## 高频变压器
天线线圈，振荡线圈，电视机天线阻抗变换器，行输出等脉冲变压器
