# Linux启动
## 进入启动控制程序的途径

1. Initramfs
2. initrd
3. /sbin/init
## init启动级别
| 级别   | 描述             |
| ---- | -------------- |
| 0    | 关机             |
| 1    | 单用户模式          |
| 2    | 无网络多用户模式       |
| 3    | 有网络多用户模式       |
| 4    | 保留为自定义，无定义时视为3 |
| 5    | 有GUI的多用户模式     |
| 6    | 重启             |
