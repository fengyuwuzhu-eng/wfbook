# 时间

[TOC]

## 文件时间
*  atime    上次被访问时间  
*  ctime    创建时间  
*  mtime    上次修改的日期和时间

## 系统时间、时区

### timedatectl

**查看当前系统时区**

```bash
timedatectl
```

**查看所有可用的时区**

```bash
timedatectl list-timezones
```

**筛选式查看在亚洲 S 开头的可用时区**

```bash
timedatectl list-timezones |  grep  -E "Asia/S.*"

Asia/Sakhalin
Asia/Samarkand
Asia/Seoul
Asia/Shanghai
Asia/Singapore
Asia/Srednekolymsk
```

**设置当前系统为Asia/Shanghai时区**

```bash
timedatectl set-timezone Asia/Shanghai
```

**设置硬件时间**

```bash
timedatectl set-local-rtc 1
```

**启用NTP时间同步**

```bash
timedatectl set-ntp yes
```
