# 键盘布局

[TOC]

## 概述

键盘布局设置控制文本控制台和图形用户界面中的布局。

## 设置

- 要列出可用的键映射

  ```bash
  localectl list-keymaps
  
  ANSI-dvorak
  al
  al-plisi
  amiga-de
  amiga-us
  ...
  ```

- 显示 keymaps 设置的当前状态

  ```bash
  localectl status
  
  ...
  VC Keymap: us
  ...
  ```

- 要设置或更改默认系统 keymap

  ```bash
  localectl set-keymap us
  ```
