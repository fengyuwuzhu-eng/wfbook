# 别名

[TOC]

**实现方式：**函数、alias命令。

## alias

```bash
alias new_command='command sequence'
```

为永久生效，放入`~/.bashrc`中。

**删除别名：**

1. 从`~/.bashrc`中删除。
2. `unalias`命令

**示例：**

```bash
alias rm='cp $@ ~/backup;rm $@'
```
