# 汇编语音
## 寄存器

| 寄存器 | 英文名称            | 名称        | 备注 |
|-------|-------------------|-------------|-----|
| AX    | accumulator       | 累加寄存器    |     |
| CX    | counter           | 计数寄存器    |     |
| DX    | data              | 数据寄存器    |     |
| BX    | base              | 基址寄存器    |     |
| SP    | stack pointer     | 栈指针寄存器   |    |
| BP    | base pointer      | 基址指针寄存器 |     |
| SI    | source index      | 源变址寄存器  |     |
| DI    | destination index | 目的变址寄存器 |     |
