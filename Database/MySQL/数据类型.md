### ENUM

    属性名 ENUM('值1'，'值2'，...'值n')

属性名为字段的名称。取值范围为列表中的值。每次只能取一个值。取值列表中最多有65535个值。如果加上了NOT NULL属性，默认值为取值列表的第一个元素。不加NOT NULL属性，将允许插入NULL，而且NULL为默认值。

### SET

    属性名 SET('值1'，'值2'，...'值n')

类似于ENUM，每次可以取列表中的几个值。
