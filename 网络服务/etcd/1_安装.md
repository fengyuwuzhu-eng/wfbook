# 安装

[TOC]

在本地安装、运行和测试 etcd 的单成员集群：

1. 从预构建的二进制文件或源代码安装 etcd。

   **重要提示**: 请确保执行安装说明的最后一步，以验证是否 `etcd` 在您的路径中。

2. 运行 `etcd` ：

   ```bash
   $ etcd
   {"level":"info","ts":"2021-09-17T09:19:32.783-0400","caller":"etcdmain/etcd.go:72","msg":... }
   ⋮
   ```

   **Note**: `etcd` 生成的输出是日志 — 可以忽略 info 级日志。
