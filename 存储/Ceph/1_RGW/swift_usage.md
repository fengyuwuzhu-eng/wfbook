## 安装Swift客户端

```bash
apt-get install python-pip
pip install python-swiftclient
```

## Swift命令

```bash
swift -V 1.0 -A http://localhost/auth -U mona:swift -K secretkey post example-bucket
```

```bash
swift -V 1.0 -A http://localhost/auth -U mona:swift -K secretkey list
```
